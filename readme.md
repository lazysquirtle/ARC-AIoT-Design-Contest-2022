## Introduction
自從肺炎爆發以來，口罩變成了所有人的必備品，
如此平凡的東西，也可以變的很不同。  
本作品想為普通的口罩增添功能，將其成為可以偵測咳嗽訊號，以及提示人們該丟棄與否的智能口罩。


## HW/SW Setup
HW 
>ARC AIoT EM9D DK + SHT31 + BM1383aglv + MPU9250 + OLED Display  

SW: 
>Synopsys SDK v22.01
